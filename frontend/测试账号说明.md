1. **编译小程序**

   - 在微信开发者工具中点击"编译"
   - 确保所有文件都已保存

2. **登录测试**

   - 使用上述任一测试账号登录
   - 体验完整的登录流程

3. **界面浏览**

   - 查看首页的简洁设计

   - 测试下拉刷新功能

4. **功能测试**
   - 查看成绩列表和筛选功能
   - 浏览课程表和今日课程
   - 体验数据统计页面

## 🎨 UI 设计特色

- **灰白色调主题** - 简洁优雅的配色方案
- **卡片式布局** - 清晰的信息层次
- **统一的圆角和阴影** - 现代化的视觉效果
- **响应式设计** - 适配不同屏幕尺寸
- **流畅的动画** - 提升用户体验

## 💡 注意事项

- 测试模式下的数据都是模拟数据
- 所有 API 请求都被拦截并返回模拟数据
- 登录后的数据会根据不同账号略有不同
- 如需连接真实后端，可在 `app.js` 中将 `isTestMode` 设为 `false`

## 🔄 恢复正常模式

如果需要连接真实的后端 API，请修改 `frontend/app.js`：

```javascript
globalData: {
  // ...其他配置
  isTestMode: false; // 改为 false
}
```

现在您可以使用任何一个测试账号来完整体验新的 UI 设计了！🎉
